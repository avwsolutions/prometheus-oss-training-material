# Prometheus OSS - Modern Observability with Prometheus and OpenTelemetry training material

Prometheus and OpenTelemetry together provide a solid foundation to set up open source (oss) based Modern Observability. The combined solution provides ways for collecting, monitoring, analyzing, and correlating telemetry data across modern distributed systems, in particular Metrics.

Combined with Grafana (oss) it can provide a Unified platform which completes visibility in all Observability pillars like Metrics, Logs and Traces.  Grafana (oss) is here to help you collect, explore, query, visualize, and analyze telemetry data effectively. These technologies empower you to gain deep operational insights, detect anomalies, troubleshoot performance issues and improve the reliability and performance of your applications and infrastructure.

During this course, the attendee learns about the fundamental skills needed to understand the concepts of modern Observability and start building an Observability platform using Prometheus, OpenTelemetry and Grafana. With a deep focus on Metrics, participants will learn how to collect them using Exporters, analyze them using PromQL, visualize them in Grafana dashboards and send out some actionable alerts.

Main topics are mentioned in the [Labs](labs/README.md).
 
The hands-on part is fully scenario based, including helpful [lab exercises](labs/README.md).
